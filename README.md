# SUSE 15 安裝示範

## 步驟 1：下載安裝映像檔

前往 SUSE 官網，下載對應硬體設備之安裝檔。

本次教學以下載15 SP7為例

![如圖](image/suse17.jpg)

![如圖](image/suse17_2.jpg)

並準備 **SLE‑15‑Packages-x86_64‑GM‑DVD1.iso** 作為軟體源。

---

## 步驟 2：開機並啟動安裝程式

- 將映像掛載為 DVD 或寫入 USB 後重開機。
- 選擇 **Installation** 選項進入安裝流程。
- 安裝過程中若出現 "Validation Check Failed" 提示，選擇 **Yes**（可能需多次）。

---
